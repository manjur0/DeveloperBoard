Different Ways to Select an Element in the DOM
document.getElementById("id") → Selects an element by its id.
document.getElementsByClassName("class") → Selects elements by their class.
document.getElementsByTagName("tag") → Selects elements by their tag name.
document.querySelector("selector") → Selects the first matching element using CSS selectors.
document.querySelectorAll("selector") → Selects all matching elements using CSS selectors.
Difference between innerHTML, innerText, and textContent
(1) innerHTML Returns or sets the HTML content inside an element, including HTML tags.
(2) innerText Returns or sets only the visible text, ignoring hidden elements.
(3) textContent Returns or sets the entire text content, including hidden elements.
What is Event Delegation in the DOM?
Event delegation is a technique where a parent element handles events for its child elements, even if they are added dynamically. Instead of adding event listeners to each child element, we add a single event listener to the parent.
What is Event Bubbling in the DOM?
Event bubbling means that when an event occurs on a child element, it first triggers on that element and then propagates upwards to its parent elements.
